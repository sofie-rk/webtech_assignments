Questions

  1. What does the server in the client server architecture do? The server is a piece of software that recieve, process and respond to requests from the client.
  2. What does the client do? Client is a software or application that request and recieve service from a server.
  3. What is an IP adress? IP (internet protocol) address is a unique numeric identifier assigned to each device on the internet (which uses Internet Protocol for communication). The IP address consits of four parts, and each part is a number between 0 and 255.
  4. What is DNS? Most Internet host or servers have a domain address in addition to an IP address. This is easier to use and remember since the domain is words and not a unique number. DNS (Domain Name System) maps the domain names to their IP address. In this way, the user can use the domain name instead of the IP address.
  5. What is the difference between the Internet and WWW? Internet is a global network of networks, consisting of connected computers and other hardware. WWW (World Wide Web) is where information exchange on the Internet (or by global computer networks). By using a metaphor; if the internet is connecting roads and towns, the WWW will be the thing you see such as trees, cars and houses.
  6. Explain what HTML is. HTML (Hyper Text Markup Language) is the standard publishing language by the World Wide Web. It describes the structure of a page, by using elements and tags.
